# Top Score beginner roadmap

This project is a native Android Studio Kotlin app that uses Jetpack Compose, MVVM-style view models, the Navigation component, and Material 3.

## What exists now

- Splash screen
- Login screen
- Create account screen
- Forgot password screen
- Home screen placeholder with profile drawer, search field, created groups, joined groups, and create-group action
- Firebase-ready repository and configuration placeholders

## Next milestones

1. Connect Firebase Auth, Firestore, and Storage.
2. Replace the in-memory repository with Firebase repositories.
3. Build group setup screens C1-C4.
4. Build admin group scoring screens C5-C12.
5. Build viewer/student screens D2-D11.
6. Add offline/internet status detection.
7. Add Hindi/English app language resources.
8. Add Play Store release assets, privacy policy, and signed release build.
